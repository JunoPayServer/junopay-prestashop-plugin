# JunoPay PrestaShop Module

PrestaShop 8 payment module for Juno Pay Server.

## Install

Copy `modules/junopay` into a PrestaShop installation, then install and enable the `junopay` module from the PrestaShop admin module manager.

Configure:

- API base URL
- Merchant API key
- Zatoshis per currency unit

The module creates a Juno Pay Server invoice during checkout and redirects the customer to a hosted invoice page inside PrestaShop.
